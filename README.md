# CH557 程序集

本仓库是学习CH557 MCU的程序.

项目文件如下

```
CH557_APPS
├─examples    //EIDE 自带例程，可参考
│  ├─adc
│  ├─blink
│  └─ws2812
│  └─。。。。。。
├─peripheral_lib  //SDCC格式的 WCH 官方库函数
│  ├─include
│  └─source
├─User_examples   // 用户的项目代码存放处，每个项目以文件夹区分开
│  └─1-1-IIC驱动SSD1306
│      ├─Inc
│      └─Src
├─User_Lib        //  // 用户的项目代码封装完的库存放处
└─WCH_documents
    └─CH557EVT    //可从WCH官方下载到的资料
```
